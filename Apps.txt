Para Android

Una vez hecho cambios:

npm run build
npx cap sync

Y para abrir android studio:
npx cap open android

Para subir esto:
En Android Studio: Build > Generate Signed Bundle / APK.
Selecciona Android App Bundle (.aab).




Para iOS

npm run build
npx cap copy
npx cap open ios


Deploy:
- Selecciona el destino: En la barra superior de Xcode, donde eliges el simulador, selecciona "Any iOS Device (arm64)"
- Product > Archive.
-Distribute App
-Custom
-Export
